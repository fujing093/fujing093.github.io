查找和排序

查找

  1. 顺序查找
  2. 二分查找（代码）
  3. 哈希表查找
  4. 二叉排序树查找
    二分查找：在排序的数组（或部分排序的数组）中查找一个数字或统计某个数字出现的次数。
    哈希表查找：优点：O(1)时间内查找，效率最高；缺点：需要额外空间实现哈希表。

排序

  1. 插入排序
  2. 冒泡排序
  3. 归并排序
  4. 快速排序（代码）递归、非递归 总体的平均效率最好，但不是最优。
    优劣、复杂度、场合、额外空间消耗、平均时间复杂度、最差时间复杂度
